Concept 2 Logbook | Personal API token

Getting the personal API token

  1. Create or log into your account at log.concept2.com.
  2. Go to Profile > Edit Profile.
  3. Click on Applications and scroll down to Concept2 Logbook API.
  4. Click Connect to Concept2 Logbook API. You will be asked to authorize access to your account.
    Click Approve Concept2 API.
  5. On the next screen, you will see your access token, which can be used to read your results via the API.

For more information, see https://log.concept2.com/developers/documentation/ .
